# /init — Project Initialization

// turbo-all

## Steps

1. Read `package.json` to understand dependencies and scripts
2. Read `app/layout.tsx` and `app/page.tsx` to understand the app structure
3. Read `next.config.ts` and `tsconfig.json` for configuration details
4. Check for any `.env` or `.env.local` files
5. Run `git log --oneline -10` to see recent commit history
6. Run `git status` to check for uncommitted changes
7. Summarize the project overview (tech stack, structure, current state) back to the user
